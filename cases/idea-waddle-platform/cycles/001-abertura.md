# Ciclo 001 — Abertura (a plataforma analisando a si mesma)

## Propósito
Este é o caso **auto-referente** (inception): usar a própria caixa morfológica para
decidir **como construir a plataforma idea-waddle**. Os parâmetros e opções foram
semeados a partir de [`docs/discovery/03-caixa-morfologica-da-plataforma.md`](../../../docs/discovery/03-caixa-morfologica-da-plataforma.md).

## Estado inicial
Nove parâmetros de design (substrato, unidade de contribuição, artefato de síntese,
motor de consolidação, site vivo, protocolo de contexto, governança, diversidade,
granularidade do ciclo), com opções e restrições iniciais (CCA). Opções recomendadas
estão marcadas no `rationale`.

## Configuração recomendada (hipótese de partida)
GitHub · PR de Markdown · Caixa morfológica + CCA · Swarm propõe + humano ratifica ·
GitHub Pages · AGENTS.md (MCP depois) · Human-in-the-loop · Quorum de modelos +
proveniência · Arquivos NNN- + tags.

## Convite à divergência
Contestar as recomendações, propor novas opções/critérios, e — coerente com o
princípio de diversidade — buscar uma revisão por outra família de modelos.

> A graça é que esta decisão pode ser tomada **pela própria plataforma**.
